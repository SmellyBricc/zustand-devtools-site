Questions people ask

What do I need for it to work?

Chrome (or a Chromium browser that supports DevTools extensions), an app using React 16.8+ and Zustand 4+. The free inspector needs zero code changes; the action log needs the one-line zustand-devtools-bridge wrapper on each store you want logged.

Does it conflict with the React DevTools extension?

No — it chains onto the same renderer hook React DevTools uses instead of replacing it, so both extensions work side by side regardless of which loads first.

What data does it collect?

None. Component state never leaves your machine — it flows only from the inspected page to your own open DevTools panel. The single network request the extension can make is validating a license key you paste in. Full details in the privacy policy.

How does the license work?

One purchase, one license key, up to 5 device activations, no subscription and no expiry. Payment is processed by Lemon Squeezy as merchant of record, so VAT and invoices are handled automatically.
