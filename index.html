How people get it

Distributed as a Chrome extension, discovered via the Chrome Web Store and word of mouth among Zustand users.

1
Install Zustand DevTools from the Chrome Web Store (listing in progress).
2
Open DevTools on any page using Zustand — the panel works immediately, free.
3
To unlock the action log, add the zustand-devtools-bridge package to a store and purchase a license once.
